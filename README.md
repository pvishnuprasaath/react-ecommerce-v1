# React-Ecommerce-v1

Minimal e-commerce web app for fashion shopping.

### Technologies used

**FrontEnd**

- ReactJS Lib
- Redux for state management
- SCSS for styling

**BackEnd**

- NodeJS
- Strapi Headless CMS with MySQL for content management
- Stripe payment API
